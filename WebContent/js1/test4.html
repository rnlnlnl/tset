연산자

1. 산술 연산자

+ - * / %

2. 대입연산자

변수에 데이터를 정하는 연산 =(대입연산자), +=, -=, *=, /=, %= (복합 대입연산)

1 2 3

3. 증감 연산자

숫자 데이터를 1증가, 1감소 시키는 연산자 ++A / A++, --A / A--

4. 비교연산자

두개의 데이터를 비교하는 연산. 해당 연산의 결과는 boolean-true/false
> < >= <= == != === !== [>=] 와 [=>] 동일한가? 다름! => 연산의 경우는 = 대입연산이기 때문에 오른쪽에 연산만 있어서는 안됨 == : 같다, != : 다르다, ===,!== : 값과 데이터 타입이 같은가, 다른가

5. 논리 연산자

논리값(t/f)을 사용하여 연산을 처리하는 연산자 &&(AND), ||(OR), !(NOT)

6. 삼항(조건)연산자

조건식의 결과에 따라서 실행하는 코드가 변경되는 연산자 조건식? 실행문1(참) : 실행문2(거짓) ;
